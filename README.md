# Document Management System

## Overview
This project is a Document Management System that leverages Natural Language Processing (NLP) and ElasticSearch for efficient document storage, retrieval, and processing. It allows users to upload files to an AWS S3 bucket, providing a scalable solution for handling unstructured data.

## Features
- **NLP Processing**: Utilize NLP techniques to analyze and process documents.
- **ElasticSearch Integration**: Efficiently index and search documents using ElasticSearch.
- **AWS S3 Integration**: Upload and manage documents in an AWS S3 bucket.
- **Backend**: Built using Python with FastAPI and Uvicorn for handling requests.
- **Frontend**: Developed with React, utilizing Yarn for package management and development.

## Technologies Used
- **Backend**: Python, FastAPI, Uvicorn
- **Frontend**: React, Yarn
- **Cloud Storage**: AWS S3
- **Search Engine**: ElasticSearch
